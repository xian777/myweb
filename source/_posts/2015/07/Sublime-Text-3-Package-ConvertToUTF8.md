Sublime Text 3 預設不支援中文編碼，所以需要安裝ConvertToUTF8套件來解決這個問題
按下Ctrl&nbsp;+ Shift&nbsp;+ P叫出命令視窗，輸入Install Package，選擇Package Control: Install Package
