# 安裝預設版本
```
$ sudo apt-get update
$ sudo apt-get upgrade -y
$ sudo apt-get install docker.io -y
```

# 移除docker
```
$ sudo apt-get remove docker docker-engine docker.io -y
$ sudo apt-get autoremove -y
$ sudo apt-get purge -y 
```
